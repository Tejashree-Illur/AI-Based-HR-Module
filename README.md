# 🚀 SmartRecruit AI

### AI-Powered Recruitment & Human Resource Management System

SmartRecruit AI is a next-generation Human Resource Management System (HRMS) designed to automate and streamline modern recruitment workflows using Artificial Intelligence.

Built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) and powered by Groq AI, the platform helps organizations efficiently manage candidates, automate hiring decisions, schedule interviews, track employee performance, and gain actionable insights through AI-driven HR operations.

---

## 🎯 Project Objective

The objective of SmartRecruit AI is to reduce manual HR effort and improve recruitment efficiency through:

* AI-Powered Resume Screening
* Intelligent Candidate Evaluation
* Automated Interview Scheduling
* AI Interview Question Generation
* AI HR Assistant
* Employee Performance Tracking
* Role-Based Access Control

---

## ✨ Key Features

### 🔐 Multi-Role Authentication & Authorization

Supports four organizational roles:

* Management Admin
* Senior Manager
* HR Recruiter
* Employee

Each role has controlled access to system functionalities.

---

### 👥 Candidate Management

* Add Candidates
* Upload Resumes
* Track Candidate Status
* Search & Filter Candidates
* Candidate Lifecycle Management

---

### 🤖 AI Resume Screening

Recruiters can enter required skills and instantly receive:

* Match Score
* Skill Matching Analysis
* Missing Skills Identification
* Screening Recommendation

This reduces manual resume review effort.

---

### 🧠 AI Candidate Evaluation (Groq AI)

SmartRecruit AI performs intelligent candidate analysis using Groq LLM.

AI-generated insights include:

* Match Percentage
* Strengths
* Weaknesses
* Interview Readiness
* Hiring Recommendation

---

### 🎤 AI Interview Question Generator

Generate skill-based interview questions dynamically using AI.

Example Skills:

* React.js
* JavaScript
* Python
* SQL
* MongoDB
* Machine Learning

---

### 💬 AI HR Assistant

An AI-powered HR chatbot capable of answering recruitment-related questions such as:

* Show shortlisted candidates
* Show rejected candidates
* How many employees are available?
* List joined employees

---

### 📅 Interview Scheduling & Automation

Schedule interviews by:

* Selecting Candidates
* Assigning Interviewers
* Setting Interview Date & Time
* Managing Interview Status

Integrated email notifications automatically inform candidates about interview schedules.

---

### 👨‍💼 Employee Management

Selected candidates can be converted into employees.

Employee information includes:

* Employee Status
* Performance Rating
* Hiring Progress Tracking

---

### ⭐ Performance Tracking

Managers can evaluate employee performance using rating scores.

The dashboard automatically calculates:

* Average Rating
* Top Performer
* Employee Performance Metrics

---

### 📊 Dashboard Analytics

Real-time analytics provide insights into:

* Total Candidates
* Employees
* Uploaded Resumes
* Shortlisted Candidates
* Rejected Candidates
* Interviews Scheduled
* Interviews Completed
* Average Ratings
* Top Performer

---

### 📥 CSV Export

Export recruitment and employee data for reporting and analysis.
